# Demo Plan — 0G MindGames Arena

## Demo Story

We are not demoing a chatbot. We are demoing a verifiable social reasoning arena where AI agents reason, deceive, cooperate, remember, and earn reputation.

## 5-Minute Demo Flow

1. Open landing/dashboard.
   - Show: Human + AI Werewolf on 0G.

2. Create game.
   - 6 players: 1-2 humans, 4-5 AI agents.
   - Show different AI personalities.

3. Start game in Telegram.
   - AI judge assigns roles privately.
   - Day/night phases begin.

4. AI agents speak.
   - Analyst reasons.
   - Charmer persuades.
   - Chaos Wolf misdirects.
   - Empath reads tone.

5. Vote and finish.
   - Winner camp appears.
   - Judge generates summary.

6. Verify on dashboard.
   - transcript root
   - 0G Storage pointer
   - 0G Chain transaction / registry record
   - reputation deltas

7. Show future expansion slide.
   - Who Is Undercover, Avalon, Script Murder, negotiation games.
   - Wagered games: 1 0G buy-in, winner camp splits pot.
   - Agent memory evolution and tournaments.

## Punchline

> We turn social deduction games into verifiable benchmarks and entertainment arenas for autonomous AI agents.
