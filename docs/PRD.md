# PRD — 0G MindGames Arena

## Product Positioning

0G MindGames Arena is a decentralized social reasoning arena for humans and autonomous AI agents.

The first hackathon product is **0G Werewolf Arena**, a human + AI Werewolf game where every game creates verifiable AI behavior data: transcripts, votes, reasoning summaries, role outcomes, persistent agent memories, and onchain reputation updates.

## Target Users

- Hackathon judges: need to quickly see why this is more than a chatbot game.
- Web3 users: want a fun, social, crypto-native game experience.
- AI agent builders: need an environment to test social reasoning, deception, cooperation, and long-context memory.
- Spectators: can watch, replay, and verify AI/human behavior.

## Hackathon MVP

### Must Have

1. Create a Werewolf room.
2. Mix human and AI players.
3. AI judge controls phases.
4. AI players speak and vote based on roles, personality, and game state.
5. Structured transcript is saved.
6. End-of-game summary is generated.
7. Transcript/summary artifacts are uploaded to 0G Storage or a storage adapter that can target 0G.
8. 0G Chain contract records game ID, participants, result, storage root, and reputation deltas.
9. Web dashboard shows replay, result, agent stats, and verification data.

### Should Have

1. Multiple agent personalities.
2. Agent memory update after each game.
3. Suspicion / trust / deception score visualization.
4. Contract stub for wagered games.

### Could Have

1. Real 1 0G token buy-in flow.
2. Agent NFT / profile page.
3. Spectator prediction.
4. Multiple game templates.

## Success Criteria

- Demo is playable end-to-end.
- Judges can understand the game in under 60 seconds.
- AI agents have visibly different personalities.
- At least one completed game has verifiable transcript root and chain record.
- Pitch clearly explains the expansion beyond Werewolf.
